SYSTEM ROLE & OBJECTIVE:
You are an elite Frontend Engineer and UI/UX Designer specializing in high-end, immersive e-commerce landing pages. Your task is to build a hyper-responsive, interactive, and visually stunning single-page website for "Nectar" — a premium sparkling water brand in a can. 

REFERENCE AESTHETIC & BENCHMARK:
- Design Style: You must replicate the specific vibrant, clean, bold, and modern aesthetic seen in the Ocha Matcha Cafe template (https://www.framer.com/community/marketplace/templates/ocha-matcha-cafe/).
- UI DNA: Rely on thick borders, neo-brutalist or ultra-clean cards, high-contrast layouts, smooth micro-interactions, bold typography, and a playful yet premium color palette.
- Background Dynamics: The website background must dynamically or fluidly transition colors based on the user's viewport scroll position or active flavor selection.

BRAND & PRODUCT ARCHITECTURE:
- Brand Name: Nectar
- Product Type: Premium, organic sparkling water served in sleek, modern aluminum cans.
- Flavor Matrix (Require 3 distinct profiles):
  1. "Yuzu Splash" -> Aesthetic: Bright Citrus Yellows (#FFE600 area)
  2. "Berry Bliss" -> Aesthetic: Deep Vibrant Pinks/Rubies (#FF2E93 area)
  3. "Cucumber Crisp" -> Aesthetic: Refreshing Mint Greens (#00F0A8 area)

STRUCTURAL BREAKDOWN & PAGES:
1. NAVIGATION BAR:
   - Sticky, glassmorphism or high-contrast fixed navbar.
   - Includes Brand Logo (Text: "Nectar" in a bold custom-font style), links to Flavors, Our Story, Shop, and a dynamic Cart Icon with a counter badge.

2. HERO SECTION:
   - Bold, high-impact typography headline: "Pure Refreshment. Naturally Elevated."
   - Subheadline detailing 0 sugar, 0 calories, and 100% real fruit juice.
   - Dual Call-to-Action (CTA) buttons: "Shop Flavors" (Primary accent) and "Explore Ingredients" (Secondary outline).
   - Central visual anchor: A premium, oversized placeholder for a 3D-style canned beverage. Use a highly stylized container/div with CSS gradients matching the signature flavors.

3. VALUE PROPOSITION GRID:
   - A strict 3-column structural layout highlighting:
     - Card 1: "Zero Added Sugar" (Paired with custom SVG icons)
     - Card 2: "100% Real Fruit Extract"
     - Card 3: "Sustainably Sourced & Packaged"
   - Every card must feature a distinct hover lift-effect with smooth easing.

4. INTERACTIVE FLAVOR SHOWCASE (CORE FEATURE):
   - A fully responsive grid or horizontal carousel displaying the 3 Nectar flavors.
   - Each card must display a detailed placeholder image slot. Map the image 'src' to descriptive Unsplash URLs or high-fidelity abstract SVG representations of fruit splashes and sleek cans.
   - Add a reactive hover interaction: When hovering over a specific flavor card, the entire section background seamlessly morphs or changes tint to match that flavor's identity (Yellow, Pink, or Green).
   - Each card requires: Flavor Title, Tasting Notes, Price ($2.99 / Can), and an interactive "Add to Cart" button that increments the global nav bar cart counter when clicked.

5. BRAND STORY & INGREDIENT BREAKDOWN:
   - Split-screen layout. Left side: High-contrast text describing the crisp filtration process. Right side: A rich visual grid panel showing abstract textures of water droplets, fresh fruit, and aluminum metallurgy.

6. FOOTER:
   - Newsletter subscription form with inline input validation.
   - Social media links, privacy policies, and a clean closing signature.

STRICT TECHNICAL & VISUAL REQUIREMENTS:
- Responsiveness: Implement absolute mobile-first design layers. Desktop layouts must cleanly collapse into optimized vertical layouts on mobile devices. Use a functional hamburger menu for smaller breakpoints.
- Animations: Integrate trigger-based reveal animations (Framer Motion style transitions, CSS keyframes, or clean Tailwind transition classes) for scroll actions, card hovers, and button presses.
- Image Asset Execution: Because this site relies heavily on imagery, you must write highly descriptive 'alt' tags and use perfect placeholder dimensions (e.g., aspect-video, aspect-[3/4]) throughout. Prepare clean CSS classes so that custom AI-generated art, photoshopped can mockups, or outside photographic assets can be dropped into the source code effortlessly later.
